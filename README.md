# Ro-FPS
A program built on Python that edits line 22 in GlobalBasicSettings_13.xml in the Roblox client (&lt;int name="FramerateCap">-1&lt;/int>) to change the framerate cap. This was created because it gets annoying manually editing it every time, so why not just build a program to do it quicker and simpler? And I'm sharing it with you! :)
(You MUST have Roblox closed when you click apply or it will NOT work.)
